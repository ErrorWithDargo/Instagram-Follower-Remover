# Instagram Follower Remover

A simple Python GUI tool to automate removing followers on Instagram.

## Features
- **GUI Interface**: Easy to use interface.
- **Auto Navigation**: Goes to your profile automatically.
- **Controlled Speed**: Removes followers at a set speed (~1000/hr).
- **Safety**: Uses Selenium to simulate a real browser interaction.

## Installation
1.  Install Python.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage
1.  Run the tool:
    ```bash
    python main.py
    ```
2.  Enter your **Instagram Username**.
3.  Enter the **Number of followers** to remove.
4.  Click **"Open Browser"** and log in manually.
5.  Click **"Start Removing"**.

> **Warning**: Use with caution. Removing too many followers too quickly can get your account temporarily action-blocked by Instagram.
